# 税控设备/金融硬件安全 — UKey 日志泄露与攻击面

> Level 0: 基础原理 / 协议安全
> 基于实战发现：Win10 C盘 `_taxUKeylog` 目录含税控 UKey 日志

---

## 一、税控 UKey 概述

### 1.1 什么是税控 UKey

税控 UKey（通常指 **航信税控** / **百望税控** 等）是中国税务系统使用的 USB Key 设备，用于：

- 企业发票开具与管理
- 税务申报数字签名
- 税控数据的加密存储

### 1.2 常见品牌

| 品牌 | 产品 | 特征 |
|:-----|:------|:------|
| **航天信息 (Aisino)** | 金税盘/税控盘 | `skfp.exe` |
| **百望股份** | 百望税控盘 | `bw` 相关进程 |
| **中税** | 中税税控 | |

## 二、日志泄露风险

### 2.1 日志文件内容

从 Win10 发现的 `C:\_taxUKeylog\UKey_2025-01-21.log`：

典型的税控日志可能包含：

```
[2025-01-21 09:15:23] UKey 连接: SN=XXXXXXXXXX
[2025-01-21 09:15:25] 验证 PIN 成功
[2025-01-21 09:16:01] 发票开具: 发票代码 XXXXXXXXXX 发票号码 YYYYYY
[2025-01-21 09:16:02] 电子签章完成
[2025-01-21 09:16:05] 上传税务系统成功
```

### 2.2 日志泄露危害

| 泄露类型 | 风险 |
|:---------|:------|
| UKey 序列号 | 唯一标识，可关联企业 |
| 发票信息 | 收入金额、客户信息、交易明细 |
| 税控软件版本 | 可能包含已知漏洞 |
| 系统路径 | 了解软件安装目录和配置 |
| 错误信息 | 可能包含调试信息/路径泄露 |

### 2.3 更严重的风险

税控 UKey 本身的攻击面：

```
UKey 攻击面
├── PIN 绕过
│   ├── 默认 PIN: 12345678 / 88888888
│   └── 弱 PIN 暴力破解（无锁定机制或锁定可重置）
│
├── 数字签名滥用
│   ├── 获取 UKey 后可签发假发票
│   └── 数字签名可被重放
│
├── 驱动程序漏洞
│   ├── 内核级驱动 → 提权/沙箱逃逸
│   └── 通信协议未加密 → 中间人攻击
│
└── 物理安全
    ├── UKey 未拔出 → 直接利用
    └── 智能卡芯片提取 → 物理分析
```

## 三、金融/银行 UKey 安全

### 3.1 银行 UKey 类型

| 类型 | 代表产品 | 特点 |
|:-----|:---------|:------|
| **USB Token** | 飞天诚信/握奇 | 证书存储在硬件中 |
| **智能卡** | 国内主流 | 需读卡器 |
| **手机 OTP** | 动态口令 | 无硬件依赖 |
| **蓝牙 UKey** | 新型 | 无线连接 |

### 3.2 常见攻击面

```bash
# 1. 数字证书导出
# 如果 UKey 允许证书导出 → 可复制到其他机器

# 2. 会话劫持
# 在已登录的机器上 → 可操作银行系统

# 3. 键盘记录
# UKey PIN 输入时 → 记录 PIN 码

# 4. 中间人攻击
# 如果通信未加密或证书验证不严格

# 5. 远程桌面利用
# 远程桌面连接时 UKey 通常重定向
```

### 3.3 我们的实战

在 Win10 渗透过程中发现了 `C:\_taxUKeylog` 目录：
- 税控软件 `skfp.exe`（航信税控发票系统）
- 日志文件记录了 UKey 操作
- 日志日期为 2025-01-21（距今已有时日）

**当前利用价值有限**，但揭示了：
- 该机器用于企业税务处理
- 工作人员使用 Windows 管理税控流程
- 日志系统缺乏保护（明文保存在 C 盘根目录）

---

## 四、防御建议

| 措施 | 针对性 |
|:----|:--------|
| 不将日志保存到未加密的明文文件 | 防日志泄露 |
| UKey 随用随拔 | 防物理滥用 |
| 启用日志加密 | 即使泄露也无法读取 |
| 税控专用隔离环境 | 不与开发/办公共用机器 |
| 定期检查 UKey 接入设备 | 防未授权使用 |
